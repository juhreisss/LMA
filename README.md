# LMA
Repositório criado para atividade do senaiEcommerce.

A Be Glow é um e-commerce focado em oferecer produtos de skincare de alta qualidade, desenvolvidos para melhorar a rotina de cuidado e promover uma pele mais saudável e iluminada. A proposta da marca é unir simplicidade, funcionalidade e bem-estar, criando uma experiência digital acolhedora e eficiente para o usuário.
Objetivo do projeto:
O objetivo do projeto é criar uma plataforma digital moderna, intuitiva e acessível que permita aos usuários conhecer, explorar e adquirir produtos de skincare com facilidade. Além disso, o site busca transmitir credibilidade, organização e uma identidade visual coerente com a proposta da marca Be Glow.


Site criado com as seguintes tecnologias:
Html e css
